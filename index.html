Welcome to Travis CI

Travis CI is an open-source, distributed build system for the open source community.

We are going to blog more about it later but for now you can read up on the initial ideas on Sven’s blog and flip through a presentation that we gave at 3 conferences and 2 Ruby user groups this summer.

If you feel like this is a great project then jump into #travis on irc.freenode.net and say hi!

Don’t forget to check out the Travis Blog!
